# WingStar Web Builder — 세션 2 초기화 문서

## 세션 1 요약

세션 1에서 구축된 핵심 기능:
- Electron + TypeScript + Vite 기반 데스크탑 앱
- WYSIWYG 캔버스 (iframe 기반, 선택/더블클릭 편집/드래그 이동)
- Live/Design 모드 토글
- 멀티 페이지 프로젝트 관리 (생성/전환/삭제/내보내기)
- Monaco Editor 통합 코드 편집기
- SVG/EPS 파서 및 벡터 레이어 트리
- 속성 검사기 (스타일, SVG fill/stroke/opacity)
- 패널 드래그 리사이징
- SVG/EPS → HTML 자동 컴파일 변환
- 라벨 드래그 앤 드롭 Flow 기반 요소 재배치
- DevTools 자동 오픈 제거

## 남은 문제 및 개선 포인트

1. **SVG 임포트 버그**: 폭죽 효과만 뜨고 실제 임포트 안 됨 (타이밍 경쟁 조건, 피드백 루프)
2. **UI 버퍼링**: 비동기 처리 부족으로 SVG 임포트 시 UI 블로킹 발생
3. **WYSIWYG 조작감**: 아직 불안정한 부분 존재 (리사이즈, 드래그)
4. **EPS 파서**: 완전한 PostScript 스펙 커버 안 됨, 복잡한 EPS 파일에서 오류
5. **성능 최적화**: Monaco ↔ Canvas 양방향 동기화 리소스 사용량 개선 필요
6. **UI/UX**: 전체적으로 다듬기 필요 (한글 지원, 툴팁, 접근성)
7. **테스트 부재**: 아직 자동화 테스트 없음
