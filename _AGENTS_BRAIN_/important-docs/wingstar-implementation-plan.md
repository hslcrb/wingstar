# WingStar → Figma급 디자인 툴 로드맵

## 목표
WingStar를 HTML WYSIWYG 편집기를 넘어, 피그마/일러스트레이터 대용으로 사용 가능한
**하이브리드 디자인 & 코드 툴**로 진화시킨다.

## 핵심 원칙
- **모든 포맷은 오직 HTML** — 저장, 내보내기, 임포트 모두 HTML만 사용 (JSON/이진 포맷 금지)
- **문서 저장 위치** — 세션 폴더 (`_AGENTS_BRAIN_/sessions/s<N>/`)
- **중요 문서** — `_AGENTS_BRAIN_/important-docs/`에도 함께 저장
- **브랜치 전략** — `feat/figma-like-designer` 브랜치에서 개발, 머지는 명시적 명령 시에만
- **본질** — HTML WYSIWYG 편집기이되, 피그마/일러스트레이터처럼 사용 가능해야 함

## 단계별 로드맵

### Phase A ✅: 캔버스 드로잉 & 자유 배치 모드
- 선택/이동 도구 + 도형 생성 도구 (사각형, 원, 선)
- SVG 컨테이너 기반 실시간 프리뷰
- 도형 생성 시 SVG 엘리먼트로 캔버스에 추가

### Phase B ✅: 레이어 패널
- 모든 DOM 요소를 트리로 표시
- 보이기/숨기기, 잠금, Z-index 컨트롤
- 선택 연동 (캔버스 ↔ 레이어 패널)

### Phase C ✅: Undo/Redo 시스템
- UndoManager 클래스: HTML 스냅샷 기반
- Ctrl+Z / Ctrl+Y 단축키
- 최대 50단계 히스토리

### Phase D ✅: 정렬 & 분배 도구
- 상단/중앙/하단/좌측/중앙/우측 정렬
- 툴바 버튼 그룹 (SVG 아이콘)

### Phase E ✅: 캔버스 줌 & 팬
- Ctrl+마우스휠 줌 인/아웃
- 줌 레벨 표시 & 리셋 버튼
- CSS transform scale 기반 (25%~400%)

### Phase F ✅: 단축키 시스템
- Delete: 선택 요소 삭제
- Ctrl+Z / Ctrl+Y: Undo/Redo

### 다음 Phase (예정)

- **드래그 레이어 재정렬** — 레이어 패널 내 드래그로 순서 변경
- **스마트 가이드 / 스냅** — 정렬 가이드라인
- **멀티 선택** — Shift+클릭 여러 요소 선택
- **그룹/언그룹** — 요소 그룹화
- **SVG 패스 편집** — 펜 도구, 앵커 포인트 조작
- **컴포넌트 시스템** — 디자인 컴포넌트, 심볼, 인스턴스
- **Copy/Paste** — Ctrl+C/V 요소 복사/붙여넣기
- **그리드/레이아웃 도구** — 가이드, 그리드 오버레이
- **HTML-only 프로젝트 저장/열기** — 페이지 + 스타일을 단일 HTML로 저장
